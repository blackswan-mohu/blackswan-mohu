## 🦢 关于我

> **「黑天鹅」** —— 那些不可预测、却影响深远的稀有事件。名字取自 Nassim Taleb 的同名之作。
> 我信奉在不确定性中成长，用代码去拥抱每一个 *improbable*。

```yaml
name:      墨鹄 / blackswan
role:      全栈开发者
style:     Vibe Coding 🎧      # 跟着感觉与心流写代码
mindset:   终身学习 📚         # 综合型选手，永远保持好奇
motto:     "拥抱不确定性，把黑天鹅变成机会"
```

- 🧩 **综合型选手**：前端到后端、设计到部署，什么都想搞明白
- 🎧 **Vibe Coding**：戴上耳机、进入心流，让直觉与工程一起工作
- 📚 **终身学习**：对新技术保持好奇，永远在 `git pull` 新知识
- 🦢 **反脆弱**：越是波动，越要成长
- 💬 找我聊：技术、产品、金融与不确定性的一切
