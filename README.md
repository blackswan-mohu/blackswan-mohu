- 🎧 **Vibe Coding**：戴上耳机、进入心流，让直觉与工程一起工作
- 📚 **终身学习**：对新技术保持好奇，永远在 `git pull` 新知识
- 🦢 **反脆弱**：越是波动，越要成长
- 💬 找我聊：技术、产品、金融与不确定性的一切
